# Ficha Owlbear — Versão Avançada (Anime)

Arquivos prontos para hospedar. Instruções rápidas:

1. Faça upload da pasta para um host estático (GitHub Pages / Netlify / Vercel).
2. Copie a URL pública do `manifest.json` (ex: `https://seusite.com/manifest.json`).
3. No Owlbear Rodeo → Extensions → Install from URL → cole a URL.
4. Use o botão gerado para abrir a ficha.

## Recursos
- Barras de Vida/Mana/Stamina com botões e cor.
- Técnicas/Talentos expansíveis.
- Inventário com peso total e equipamento de arma.
- Cálculos automáticos (bônus de atributo, defesa, esquiva, dano base).
- Salvamento: cena / token / localStorage (fallback).

Se quiser que eu personalize fórmulas, entenda algum campo da planilha como especial, ou gere o ZIP e o link de download, me avise.
